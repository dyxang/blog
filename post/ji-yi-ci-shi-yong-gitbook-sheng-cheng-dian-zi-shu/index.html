记一次使用Gitbook生成电子书（失败）

2020-08-26 2 min read # 技术 # 随笔

开始学 python 了，看到 Github 项目《Python语言基础50课》挺不错想下下来在 kindle 上看，可惜没有现成电子书，只有md文件和知乎专栏文章。于是第一次踩坑 gitbook，记录一下适合自己的操作步骤。

请注意文章部分具有的时效性

开始前

  • 我已安装 Node.js
  • 已经把 Node.js 加上了淘宝镜像源来加速
  • 我已经安装 Calibre

准备工作

  1. 安装 gitbook等所需模块：npm install gitbook-cli ebook-convert -g
  2. （使用加速）克隆需要做成书的项目到本地：git clone https://g.ioiox.com/https://github.com/jackfrued/Python-Core-50-Courses.git
  3. 提取文件夹里所有md文件的文件名，随后要用到。这里使用的是 Windows 的 tree 命令：tree E:\Python-Core-50-Courses /f >E:\Python-Core-50-Courses\SUMMARY.md
  4. 当然，你也可以直接在该文件夹下使用 tree /f > SUMMARY.md
