头像、昵称与标志的更换历史

2021-01-11 4 min read # 博客 # 生活 # 随笔

很久很久之前，我都不知道自己会在博客路上走这么远。
就简单谈一下博客/头像的历史与寓意

第一张头像

起初正式拥有独立博客时（二零一六年）我使用的是 Wordpress 注册了官方账号，为了使用 Gravatar，上传了一个头像——一个 @海猫络合物 画的人物截图，加上“爱写”两个字，寓意坚持写博客。而 @海猫络合物 在当时只是一个原画画师，还没有自己合作出明日方舟这样的爆款手游，不过有趣的是，当时 ta 微博上画的一些草稿与原型基本符合了日后明日方舟干员的形象与设定，而我截取的这一张头像刚好是贴近 ta 日后创立的明日方舟的设定图。

在那之后很长一段时间，大概四年多，我都没有专门再弄一个头像来放在博客里，直到我最后一次使用 Windows 7，在从来不用的数位板上画出来这样一个头像。很长一段时间我甚至在社交软件里都使用它，即使不是很漂亮，但有个性就足够了。

现在，我决定在之后的社交网络注册中，坚持使用途读博客的头像，提高辨识度。这头像比较极简，长这样：

其实我在社交软件上的头像基本都很极简并辨识度很高，以至于在某些社群里失踪半年再回来还有人认得我……另外我现在刚注册了一个新的长毛象账号[1]，欢迎关注~老账号几年前注册的一直没用，就忘了🌚。

网站图标来由

第一个网站图标已经消失在互联网中了，是一个简单的 D 字母与各种寓意的拼接，其实并算不上真正意义上的网站图标，真正意义上我自己设计的网站图标一直沿用至今，没错，就是这个：

不出意外的话我不会再更改了，类似变身器（大雾）的图形其实是字母“Y”，也是我姓名中的缩写，而白色方块则代表着字母“D”也就是我昵称的首字母，然后整体呈火炬状。制作它的起因就是在 Farbox 至 Bitcron 时期（参见 关于我 > 时间轴），如果你没有自己的网站图标的话，默认显示的是服务商的官方图标，这一点很恼火，所以我就自制了一个图标。

昵称

起初我的博客叫 ③d's blog，原因嘛，就是之前的昵称连续带三个“d”，多说几句，那个时候因为是博客独立域名，活着的友链还蛮多，甚至莫名其妙被一个独立博客收录站给收录过😂[2]。
之后，我注册了 Github 账号，结合名字创立了“Dyxang”这个英文名，我的读法是“迪祥”，但是英语正确读法是“迪克斯昂”。至于途读，也是一直说的 to do > 2do > 途读，看我以前的域名就知道了。

  1. 关于长毛象的介绍，参见《什么是长毛象？》文档 ↩︎

  2. 让人无语的是，我的博客独立域名末期，我使用的 Typecho 博客主题的博主（早已互加友链）把自己的博客主题改为收费后就把我的友链直接删除，哪怕我把收费版本之前的那套博客主题换掉再重新加他友链他还是不理我，我想不至于我仅仅是使用收费前的主题版本让他不想让别人知道吧，而且有话好好说嘛，人心难料啊。 ↩︎
