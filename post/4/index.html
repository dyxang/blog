特点：只提供配置，不提供资源。尽量准确，多注释，好理解，针对小白。
#换软件镜像源#

为了加快下载软件速度和更新速度

  1. 阿里云镜像源
  2. 新浪镜像源
  3. 各大学镜像源
pi@raspberrypi:~$ sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak   #备份为sources.list.bak
pi@raspberrypi:~$ sudo nano /etc/apt/sources.list      #用nano编辑sources.list  文件

删除原有的内容或者用#注释掉原来的源,添加镜像源.

示列：
deb http://mirrors.aliyun.com/raspbian/raspbian/ wheezy main non-free contrib
deb-src http://mirrors.aliyun.com/raspbian/raspbian/ wheezy main non-free contrib

该例子是针对wheezy raspbian的，阿里云没有jessie raspbian，你可以在文章底部查看jessie源的例子。

然后使用 Ctrl+O 回车后保存文件，Ctrl+X 退出编辑器
pi@raspberrypi:~$ sudo apt-get update && apt-get upgrade -y #最后一步 更新 系统软件 并 更新 已安装的包

支持中文相关配置

  1. 安装中文字体。
    终端下执行命令：sudo apt-get -y install ttf-wqy-zenhei
  2. 设置显示中文。
    终端下执行命令：sudo raspi-config
    *按空格键在前面打勾或去掉勾（就是已勾选），PageUp PageDown快速翻页，Tab键跳到OK按钮上， 去掉en_GB.UTF-8 UTF-8，勾上：“en_US.UTF-8 UTF-8”“zh_CN.UTF-8 UTF-8”“zh_CN.GBK GBK” ，下一屏幕默认语言选 zh_CN.UTF-8
  3. 在命令行输入 sudo reboot

安装输入法

  • 基于 Fcitx 输入法框架—— google 拼音输入法
    命令行：
sudo apt-get install fcitx fcitx-googlepinyin fcitx-module-cloudpinyin fcitx-sunpinyin
  • SCIM——支持多国语言的输入法
    命令行：
sudo apt-get install scim-pinyin
  • Ibus 输入法——基于 Python 开发
    命令行：
sudo apt-get install ibus ibus-pinyin
  • 云拼音输入法-应该有云词库
    命令行：
sudo apt-get install fcitx-cloudpinyin
  • 一套完整的输入法安装（未经测试）
    命令行:
sudo apt-get install fcitx fcitx-sogoupinyin fcitx-config-gtk fcitx-frontend-all fcitx-module-cloudpinyin fcitx-ui-classic
