如何更舒适使用商家已跑路的口袋阅？

2021-08-17 3 min read # 技术 # 随笔

本文不教小白，只是给动手能力强的人一个思路。虽然口袋阅商家已经跑路，店铺早已关门，但设备还是能继续用的。

使用微信读书

  • 下载和安装
    微信读书有墨水屏版，针对墨水屏做优化，官网写的很详细。目前还是能直接从口袋阅内部点图标下载的（虽然口袋阅内置商店已经歇了）
    如果这个办法已经歇了，那就下载安装包后口袋阅连着电脑用 ADB 安装吧，或者小白用搞机工具箱，也有教程。
  • 组队无限卡
    微信读书手机端里答题、分享、被好友点赞联名卡，都会赠送无限卡天。每天还可以翻牌子，翻到免费领，也有分享免费领活动，每天哪个端看了多少分钟都会有统计，签到加上每天阅读时间可以在手机 app 里领无限读天数
    抽无限卡组队的话可以用这个工具
  • 上传本地书籍
    手机 app 和网页版都可以上传，注意文件名不能有下划线。

转移 Kindle 书籍

  • 从网页下载
  1. 登录亚马逊官网，进入管理我的内容和设备>内容
  2. 点击每本书前面的 ···，“通过电脑下载USB传输”，一本一本的下成 azw3 格式
  • 从设备转移
    在你的 Kindle 上一个个把想要的电子书下好，要 azw3 格式的，怎么做去问搜索引擎
  • 安装 Calibre（官网） 和 DeDRM tools （Github）
    都是明白人，知道怎么做，注意了：

    Users with calibre 5.x or later should use release 7.2.0 or later of the tools. Users with calibe 4.x or earlier should use release 6.8.x of the tools.

  • 进行 DeDRM 操作
    把下好且已经装上 DeDRM tools 插件、配置好 kindle 序列号的 Calibre 打开，有 DRM 的书籍直接拖进去，要是还打不开就转换一下格式。再打不开，脱 DRM 工具的官方压缩包有 Readme，看 Troubleshooting 就行了。完成后直接传到微信读书里看吧，除了自带的阅读器外，微信读书墨水屏版在口袋阅上体验很好。
